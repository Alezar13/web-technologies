Виды html теггов

Основные

  1. doctype - предназначен для определения типа документа. То есть по нему браузер понимает, что это html-документ, а не что-либо еще.
  2. html - это контейнер, который имеет основное содержимое страницы. Контейнером называется потому что у него есть парный тег (открывающий и закрывающий).
  3. head - содержит в себе служебные теги. Они предназначены для работы поисковых систем и браузеров. Обычно эта информация не видна пользователю.
  4. body - это основное тело страницы. Она содержит рабочие элементы, которые видны пользователям. Как правило, это информационный контент.

Служебные

Служебные теги предназначены для поисковых систем и браузеров. Браузеры по ним определяют тип документа и его интерпретацию. А поисковым системам важно для четкого сканирования документа и его ранжирования в выдаче.

  • !—…— - коментарий
  • !DOCTYPE> - объявление типа документа
  • head - контейнер для метаданных html-документа
  • title - заголовок / имя html-документа
  • meta - мета-данные веб-страницы
  • link - подключает внешние таблицы стилей
  • script - пподключает сценарии к странице
  • style - подключает встраиваемые таблицы стилей
  • base - базовый url-адрес, относительно которого вычисляются относительные адреса
  • noscript - секция, не поддерживающая скрипт
